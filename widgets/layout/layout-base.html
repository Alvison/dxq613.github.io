简介

在前端领域中布局一般有2种方式：

  • CSS 布局，使用CSS来布局，实现简单，一般用于框架布局。但是对于不熟悉CSS的开发人员来说有些困难；处理页面内部元素自适应时，需要一些技巧。
  • JS 布局，通过JS不断调整页面元素的宽高和位置来达到布局的目的。一般在窗口发生改变，收缩展开时使用，使用方便，单不够灵活。

本章节主要讲述JS布局，同时简单的描述了支持的布局种类

布局的使用和分类

布局的使用

BUI的布局作为控件的插件来使用，这样的好处在于：

  1. 任何控件都可以使用布局，同时不侵入控件本身的逻辑
  2. 使用灵活，各种布局方式可以嵌套使用，互不影响
  3. 可以很容易的改变布局方式

布局种类

BUI中提供了一下种类的布局插件

种类 类 描述 绝对布局 BUI.Layout.Absolute 通过绝对位置来布局，适用于位置固定不发生变化的布局场景 边框布局 BUI.Layout.Border 传统的东西南北中的布局方式，并提供展开折叠布局项的功能 伸缩布局 BUI.Layout.Accordion 可以展开收缩的布局方式 锚定布局 BUI.Layout.Anchor 跟各个边框锚定的布局方式 表格布局 BUI.Layout.Tabel 使用表格进行布局，一般适用于比较规整但是很少变化的布局项 多列布局 BUI.Layout.Columns 将布局项组织在多列 浮动布局 BUI.Layout.Flow 所有的元素进行浮动的布局方式

布局和布局项

上面的分类是基于布局内部的布局项如何定位，如何放置，如果一个控件使用了布局插件，那么控件的每一个子控件都会被一个布局项所嵌套，根据布局的种类进行定位。
