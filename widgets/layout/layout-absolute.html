简介

绝对定位是一种非常简单易于理解的布局方式，所有的布局项指定位置的绝对位置,为了用户使用方便，BUI对宽高的设置做了一些改进：

  1. 宽高支持数字和百分比的格式
  2. 宽高也支持简单的数学运算，例如 : width : '{width} * 0.2 + 100',表示宽度占整个布局宽度的20% + 100px

可以通过fit属性配置内部控件的宽高自适应

  1. fit: 'none', 内部控件不跟随布局项的宽高自适应
  2. fit: 'width',内部控件跟随布局项的宽度进行自适应
  3. fit: 'height',内部控件跟随布局项的高度进行自适应
  4. fit: 'both',内部控件跟随布局项的宽度、高度都进行自适应
