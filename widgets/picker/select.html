选项和值

对于一般应用而言，用户只需要提供选择项和默认值就能很好得模拟使用html中得select标签，所以BUI提供了BUI.Select.Select控件，使用这个控件你仅仅需要配置:

  • items : 选项，默认识别value、text
  • valueField : 对应的存放值的输入框，一般是隐藏域

配置项items 选择框的选项集合支持2种格式

  1. 数组： [{text : '1',value : '1'},{text : '2',value : '2'}]
  2. 对象： {'1':'1','2':'2'}

单选和多选

选择框默认是单选，可以通过multipleSelect属性来配置多选，多选返回的文本和值全部使用“,”分割

补充说明：

  • getSelectedText 方法：可以获取选中的文本，多选使用“,”分割
  • getSelectedValue 方法：可以获取选中的值，多选使用“,”分割
  • 如果不指定valueField也可以通过setSelectedValue方法设置默认值

使用store

选择框的数据使用异步数据时，可以配置store，会作为内部列表的选项进行数据的加载。


     *  var store = new Store({
     *    url : 'data.json',
     *    autoLoad : true
     *  });
     *  var select = new Select({
     *    render : '#s',
     *    store : store//设置了store后，不要再设置items，会进行覆盖
     *  });
     *  select.render();
